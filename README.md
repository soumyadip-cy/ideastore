# **IdeaStore - MERN app**

IdeaStore is a **MERN project** made using the **MERN Stack**.
> Render.com hosted link: ['ideastore'](https://ideastore.onrender.com)

---

### Features

1. Homepage: Shows all the ideas(notes) created by the user.
2. Create Note: A page to create a note or save an idea.
3. Navbar: Has links to the homepage and the create a note page.
